

***************************************************

	ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА: 
	
***************************************************

	*** У вас должен быть установлен Docker ***


1. В папке с ботом откройте файл config.py и скопируйте туда токен от своего бота (токен должен быть в кавычках). Сохраните файл.
 
2. В терминале, находясь в директории с файлами бота, пропишите следующую команду
(без кавычек, но с точкой):
 
'docker build .'

После этой команды Docker начнет создавать виртуальный образ бота (IMAGE) для запуска на вашем компьютере.
 	
3. Дождитесь окончания загрузки. Далее пропишите (также без кавычек):

'docker images'

Терминал выдаст вам список из строк (образов), запущенных на вашем компьютере.
Список образов будет выглядеть примерно вот так: 


REPOSITORY   TAG       IMAGE ID       CREATED         SIZE
<none>       <none>    d1a490eb79a5   5 minutes ago   997MB
<none>       <none>    0c6463d1953c   7 minutes ago   940MB
python       3.9.5     9b0d330dfd02   17 months ago   886MB


Вам необходимо по столбцу "CREATED" найти контейнер, который был создан только что (наиболее поздний по сроку создания). Это образ вашего бота. 

5. Скопируйте символы из столбца IMAGE ID в строке с образом вашего бота (из примера выше это будут символы "d1a490eb79a5"), и пропишите в терминале команду: 

 'docker run -d -p 80:80 [ваш  ID]'
 
 подставив вместо "[ваш  ID]" скопированные ранее символы.
 
 После окончания загрузки бот должен заработать. 
 
 Приятного использования!
